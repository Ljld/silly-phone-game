# Silly Phone Battle! 🤪📱

A hilarious two-player party game you play on **one phone**. Hold the phone between two players, each tapping their side of the screen. First to win 3 rounds is crowned the Silly Champion!

## 🎮 Play Now

**[Play on your phone!](https://ljld.github.io/silly-phone-game/)**

Just open the link on your phone's browser and start tapping!

## 🎲 Mini-Games

| Game | How to Play |
|------|-------------|
| **👆 Tap Race** | Tap your side as fast as you can — first to 20 wins! |
| **⚡ Quick Draw** | Wait for "GO!" then tap first — but don't jump the gun! |
| **🎨 Color Clash** | Tap your side only if your color matches the target! |
| **🤫 Hold Steady** | Press and hold for 5 seconds — lift your finger and you lose! |
| **📳 Shake Off** | Tap like crazy to fill your meter first! |

## 🛠️ Development

```bash
npm install
npm run dev
```

## 📦 Build & Deploy

The game auto-deploys to GitHub Pages on every push to `main` via GitHub Actions.

To build manually:

```bash
npm run build
npm run preview
```
